# 🎩 春晚“时间定格”魔术计算器 (2026 复刻版)

[English Documentation](README.md) | [在线演示](#) <!-- 这里稍后可以填 GitHub Pages 链接 -->

这是 **2026 年央视春晚** 邓男子表演的《惊喜定格》魔术中使用的计算器道具的 **Web 复刻版本**。

本项目基于 HTML/CSS/JavaScript 原生开发，不仅完美还原了 iOS 计算器的视觉风格，还完整复刻了魔术背后的核心算法逻辑。

## ✨ 核心功能

*   **UI 完美还原**: 1:1 复刻 iOS 原生计算器界面，包括按键反馈、字体和配色。
*   **响应式适配**:
    *   **手机端**: 全屏沉浸式显示，建议“添加到主屏幕”使用，效果以假乱真。
    *   **电脑端**: 自动显示为手机外框（卡片模式），背景虚化，方便演示与调试。
*   **魔术逻辑 (核心)**:
    *   **第一步**: 强制要求输入 **4 位数字**。
    *   **第二步**: 强制要求输入 **5 位数字**。
    *   **第三步 (障眼法)**: 在这一步，无论点击哪个数字键，屏幕上都会自动输入经过计算的“补数”。
*   **动态时间计算**: 
    *   最终结果永远等于 **“当前月+当前日+当前时+当前分”** 组成的 7 位数字。
    *   包含 **20秒进位容错逻辑**：当秒数到达 40秒时，程序会自动将目标结果预判为下一分钟，防止表演过程中的时间差导致穿帮。

## 🚀 表演流程指南

1.  **准备**: 在手机浏览器打开 `index.html`。
2.  **互动 1**: 请观众输入任意 **4 位数字**（如出生年份），点击 `+`。
3.  **互动 2**: 请另一位观众输入任意 **5 位数字**（任意杂乱数字），点击 `+`。
4.  **高潮**: 说明要进行“随机输入”，倒扣手机或不看屏幕，随意点击数字键。
    *   *秘密*: 此时程序会实时计算当前时间与前两步之和的差值，并自动填入该数值。
5.  **揭晓**: 点击 `=`，展示结果。结果将精准对应当下的日期和时间（例如 `2162227` 代表 2月16日 22:27）。

## ⚠️ 注意事项与使用贴士

1.  **功能局限性**: 此代码实现的计算器为**魔术专用道具**，除特定的加法流程外，无法进行常规数学运算（如减乘除功能均为视觉装饰）。请勿在非表演场景下作为普通计算器使用。
2.  **严格流程控制**: 程序逻辑强依赖于 **“4位数字 + 5位数字”** 的输入顺序。若输入位数不符，程序将阻止进入下一步。
3.  **隐藏提示语**: 代码默认显示顶部的步骤提示。若需要更真实的表演效果，请在 CSS 中将 `.magic-tip` 设置为 `display: none;`，但请务必熟练记忆操作步骤。
4.  **离线运行**: 建议将 HTML 文件保存至手机本地运行，无需联网，体验更佳。
